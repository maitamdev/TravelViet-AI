# TravelViet AI - API Documentation
## Hooks
- useTrips() - CRUD trips
- useBudget(tripId) - Budget tracking
- useChat() - AI chat sessions
- useCollaboration(tripId) - Members, comments, tasks
- useBookmarks() - Save itineraries
- useLikes() - Like itineraries
## Utilities
- formatVND(amount) - Currency format
- calculateTripDuration(start, end) - Trip days
- getDaysUntil(date) - Countdown
- formatRelativeTime(date) - Relative time

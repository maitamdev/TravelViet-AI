# TravelViet AI 🇻🇳

**AI-Powered Travel Planning Assistant for Vietnam**

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/maitamdev/TravelViet-AI)

## ✨ Features

- 🤖 **AI-Powered Trip Planning** - Get personalized travel recommendations
- 🗺️ **Smart Itinerary Generator** - Create detailed day-by-day plans
- 💰 **Budget Tracking** - Manage your travel expenses
- 📍 **Destination Discovery** - Explore hidden gems in Vietnam
- 👥 **Community Sharing** - Share and discover trips from other travelers
- 🔐 **Secure Authentication** - Protected with Supabase Auth

## 🚀 Quick Start

### Prerequisites

- Node.js 18+ and npm
- Supabase account (for backend)

### Installation

```bash
# Clone the repository
git clone https://github.com/maitamdev/TravelViet-AI.git

# Navigate to project directory
cd TravelViet-AI

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env and add your Supabase credentials

# Start development server
npm run dev
```

## 🛠️ Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **Backend:** Supabase (Database + Auth + Edge Functions)
- **AI:** Groq API
- **Routing:** React Router DOM
- **Forms:** React Hook Form + Zod
- **State:** TanStack Query

## 🌐 Deployment

### Deploy to Vercel

1. Push your code to GitHub
2. Import project to [Vercel](https://vercel.com)
3. Add environment variables:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_PUBLISHABLE_KEY`
4. Deploy!
